## Components

- React 17
- Webpack 5
- Babel 7
- Hot Module Replacement
- Dev + Production configurations
- Source Maps
- Environment configurations
- ESLint
- Prettier

## Installation

- `git clone git@github.com:marvinklein/react-boilerplate.git`
- `npm install`
- `npm start`
- visit `http://localhost:8080/`
